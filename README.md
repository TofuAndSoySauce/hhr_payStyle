    페이징도 동시에 해결하려고 Specification를 사용했으나 복잡한 쿼리문에서 가시성이 많이 떨어지게 되는 문제 때문에 사용을 지양해야 함.  

3. 어떤 기능을 어떤 라이브러리를 써서 어떠한 방식으로 할지 잘 몰랐음.
